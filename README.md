# Hautnah — Website-Projekt

Webseite für das Kosmetikinstitut Hautnah (Sabrina Herosé) in Wehr.

## Schnellstart für Claude Code

1. Lies `CLAUDE.md` zuerst — das ist das vollständige Briefing.
2. Lies `VERSIONS.md` für den aktuellen Stand.
3. Öffne `current/index.html` im Browser, um den aktuellen Designstand zu sehen.

## Status

## Wichtigste Regel

Erst klären, dann implementieren. Keine eigenmächtigen Design-Umbauten. Keine neuen Hauptversionen ohne Freigabe.
